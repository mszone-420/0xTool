## 🔥 What is 0xTool?

**0xTool** is an all-in-one media tool for Termux & Browser that can:
- 📥 Download from YouTube, TikTok, Instagram, Twitter, Facebook
- 🕵️ Extract EXIF metadata from images/videos
- 🔓 Test passwords for ZIP, RAR, PDF files
